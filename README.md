# Shaurya Manjkhola

Portfolio website built with Next.js, TypeScript, Tailwind CSS, and Gemini API.

Showcases projects, experience, skills, education, and competitive programming achievements through a modern interactive interface.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Gemini API
- Vercel

## Features

- AI Portfolio Assistant
- Project Showcase
- Experience Timeline
- Skills Overview
- Education & Achievements
- Competitive Programming Profile
- Responsive Design
